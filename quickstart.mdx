## Get started in three steps

Get your Xendit integration up and running quickly.

### Step 1: Set up your local environment

<AccordionGroup>
  <Accordion icon="copy" title="Get your API credentials">
    Sign up for a Xendit account and get your API credentials from the [Xendit Dashboard](https://dashboard.xendit.co).
    
    You'll need your public key and secret key to start integrating with Xendit's APIs.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Install the SDK">
    1. Install the Xendit SDK: `npm install xendit-node`
    2. Set up your environment variables with your API keys
    3. Start making your first API call!
    
    <Tip>Check our code examples for your preferred programming language.</Tip>
  </Accordion>
</AccordionGroup>

### Step 2: Make your first payment

<AccordionGroup>
  <Accordion icon="credit-card" title="Create a payment method">
    Use Xendit's API to create payment methods like virtual accounts, e-wallets, or credit cards.
    
    Choose the payment method that works best for your customers in your region.
</Accordion>
<Accordion icon="code" title="Test your integration">
    Test your integration using Xendit's sandbox environment.

    1. Use test API keys for development
    2. Create test payments to verify your integration
    3. Check webhook notifications are working properly

    <Tip>Always test in sandbox before going live!</Tip>
  </Accordion>
</AccordionGroup>

### Step 3: Go live

<Accordion icon="rocket" title="Deploy to production">
  1. Switch to your live API keys
  2. Deploy your application to production
  3. Start accepting real payments!
</Accordion>
