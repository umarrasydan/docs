# Get Payment Detail

Retrieves the details of a specific payment transaction using its unique identifier.

## Headers

<ParamField query="Content-Type" type="string" required>
  `application/json`
</ParamField>

<ParamField query="Authorization" type="string" required>
  `Basic <Base64Encode(API_KEY:)>`
</ParamField>

## Path Parameters

<ParamField path="payment_id" type="string" required>
  The unique identifier of the payment to retrieve
</ParamField>

## Response

<ResponseField name="payment_id" type="string">
  Xendit unique Payment ID
</ResponseField>

<ResponseField name="created" type="DateTime">
  Payment's creation time
</ResponseField>

<ResponseField name="updated" type="DateTime">
  Payment's updated time
</ResponseField>

<ResponseField name="reference_id" type="string">
  Merchant's reference ID
</ResponseField>

<ResponseField name="customer_id" type="string">
  Xendit-generated Customer ID
</ResponseField>

<ResponseField name="type" type="string">
  `PAY`: collects a one-time payment from a customer
</ResponseField>

<ResponseField name="currency" type="string">
  Currently supports IDR, THB, VND only
</ResponseField>

<ResponseField name="request_amount" type="number">
  The intended payment amount to be collected from the end user
</ResponseField>

<ResponseField name="country" type="string">
  Country code
</ResponseField>

<ResponseField name="payment_details" type="object">
  Payment information provided by the payment method provider
</ResponseField>

<ResponseField name="payment_details.terminal_id" type="string">
  Terminal ID
</ResponseField>

<ResponseField name="metadata" type="object">
  Key-value entries for your custom data
</ResponseField>

<ResponseField name="failure_code" type="string">
  Payment failure code
</ResponseField>

<ResponseField name="status" type="string">
  <Enum values={["SUCCEEDED", "VOIDED"]} />
  
  Payment's status
</ResponseField>

<ResponseField name="business_id" type="string">
  Xendit business ID
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET https://terminal-dev.xendit.co/v1/terminal/payments/pay_1234567890abcdef \
  -H "Content-Type: application/json" \
  -H "Authorization: Basic <Base64Encode(API_KEY:)>"
```

```javascript JavaScript
const response = await fetch('https://terminal-dev.xendit.co/v1/terminal/payments/pay_1234567890abcdef', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Basic ' + btoa('API_KEY:')
  }
});
```

```python Python
import requests
import base64

api_key = "your_api_key"
auth_string = base64.b64encode(f"{api_key}:".encode()).decode()

response = requests.get(
    'https://terminal-dev.xendit.co/v1/terminal/payments/pay_1234567890abcdef',
    headers={
        'Content-Type': 'application/json',
        'Authorization': f'Basic {auth_string}'
    }
)
```
</CodeGroup>
