# Push Operation Command

Sends operational commands to terminal devices for settlement and receipt printing operations.


## Headers

<ParamField header="Content-Type" type="string" required>
  `application/json`
</ParamField>

<ParamField header="Authorization" type="string" required>
  `Basic <Base64Encode(API_KEY:)>`
</ParamField>

## Request Body

<ParamField body="terminal_id" type="string" required>
  Terminal ID
</ParamField>

<ParamField body="type" type="string" required>
  <Enum values={["SETTLE", "PRINT_RECEIPT"]} />
  
  Command type
</ParamField>

<ParamField body="properties" type="object">
  Data required to initiate command
</ParamField>

<ParamField body="properties.order_id" type="string">
  Merchant's order id
</ParamField>

<ParamField body="properties.payment_method" type="string">
  <Enum values={["ID_INSERT_CARD", "ID_CONTACTLESS", "ID_BRIZZI", "ID_QRIS", "VN_CARD", "VN_QR", "TH_ALIPAY_PLUS", "TH_WECHATPAY", "TH_TRUEMONEY", "TH_LINEPAY", "TH_SHOPEEPAY", "TH_SABUYMONEY", "TH_MAXME", "TH_CARD", "TH_BBL_QRCS", "TH_MULTIIPP", "TH_QRCODE", "MY_QR", "MY_CARD", "MY_EWALLET"]} />
  
  Payment method
</ParamField>

## Response

<ResponseField name="id" type="string">
  Command ID
</ResponseField>

<ResponseField name="status" type="string">
  <Enum values={["ACTIVE", "COMPLETED"]} />
  
  Command status
</ResponseField>

<ResponseField name="terminal_id" type="string">
  Terminal ID
</ResponseField>

<ResponseField name="business_id" type="string">
  Xendit business id
</ResponseField>

<ResponseField name="type" type="string">
  <Enum values={["SETTLE", "PRINT_RECEIPT"]} />
  
  Command type
</ResponseField>

<ResponseField name="created" type="DateTime">
  Command's creation time
</ResponseField>

<ResponseField name="updated" type="DateTime">
  Command's updated time
</ResponseField>

## Command Types

<CardGroup cols={2}>
  <Card title="SETTLE" icon="calculator">
    Initiates settlement process on the terminal device. This processes all pending transactions and finalizes them.
  </Card>
  <Card title="PRINT_RECEIPT" icon="receipt">
    Triggers the terminal to print a receipt for a specific transaction or settlement summary.
  </Card>
</CardGroup>

## Example Request

<CodeGroup>
```bash cURL
curl -X POST https://terminal-dev.xendit.co/v1/terminal/commands \
  -H "Content-Type: application/json" \
  -H "Authorization: Basic <Base64Encode(API_KEY:)>" \
  -d '{
    "terminal_id": "TERM001",
    "type": "SETTLE",
    "properties": {
      "order_id": "ORDER123",
      "payment_method": "ID_INSERT_CARD"
    }
  }'
```

```javascript JavaScript
const response = await fetch('https://terminal-dev.xendit.co/v1/terminal/commands', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Basic ' + btoa('API_KEY:')
  },
  body: JSON.stringify({
    terminal_id: 'TERM001',
    type: 'SETTLE',
    properties: {
      order_id: 'ORDER123',
      payment_method: 'ID_INSERT_CARD'
    }
  })
});
```

```python Python
import requests
import base64

api_key = "your_api_key"
auth_string = base64.b64encode(f"{api_key}:".encode()).decode()

response = requests.post(
    'https://terminal-dev.xendit.co/v1/terminal/commands',
    headers={
        'Content-Type': 'application/json',
        'Authorization': f'Basic {auth_string}'
    },
    json={
        'terminal_id': 'TERM001',
        'type': 'SETTLE',
        'properties': {
            'order_id': 'ORDER123',
            'payment_method': 'ID_INSERT_CARD'
        }
    }
)
```
</CodeGroup>
